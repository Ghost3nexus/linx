1. システム概要

LINXは、LINE公式アカウントと連携した予約・顧客管理・会員決済の統合システムです。

お客様がLINEで体験予約
↓
AIが自動で空き確認・予約作成
↓
スタッフにLINE通知
↓
体験レッスン実施
↓
予約ステータスを「完了」に変更
↓
自動で入会案内+決済リンク送信
↓
3日後、未入会者に自動フォロー

自動 = システムが自動実行   手動 = スタッフが操作

2. ログイン方法

  1. ブラウザで管理画面URLにアクセス
  2. メールアドレスとパスワードを入力
  3. 「ログイン」をクリック
重要: ログイン情報は初期設定時にお渡しします。パスワードは定期的に変更してください。

4. スケジュール管理

スタッフごとの週間スケジュールをカレンダー形式で管理します。

営業時間の設定

  1. 画面右上の「Hours」ボタンをクリック
  2. 各曜日の営業開始・終了時間を設定
  3. 定休日は「Off」にチェック
  4. 「Save」で保存

予約枠の追加

  1. カレンダー上の空いている枠をクリック
  2. 開始時間・終了時間・サービス・担当スタッフを選択
  3. 定員（Max Participants）を設定
  4. 「Save」で保存

曜日コピー機能

「Copy Day」ボタンで、ある曜日のスケジュールを別の曜日にコピーできます。同じ時間割の曜日が多い場合に便利です。

5. スタッフ管理

スタッフの追加

  1. Schedule画面右上の「Staff」ボタンをクリック
  2. 「+ Add Staff」をクリック
  3. 名前（必須）・役職・メール・LINE User IDを入力
  4. 「追加」をクリック
LINE User IDの設定: LINE User IDを設定すると、そのスタッフの担当予約が入った時にLINEで自動通知されます。

LINE User IDの取得方法

  1. スタッフ本人がジムのLINE公式アカウントを友だち追加
  2. 何かメッセージを1通送ってもらう
  3. 管理画面の「Logs（会話ログ）」画面を開く
  4. そのスタッフのメッセージを見つけ、User ID（U始まりの文字列）をコピー
  5. Staff ManagementでそのIDを入力

出勤時間の設定

  1. スタッフ一覧の歯車アイコン（⚙）をクリック
  2. 曜日ごとに出勤可能な時間帯を設定
  3. 「Save」で保存

休みの申請

  1. スタッフの歯車アイコンをクリック
  2. 「Day Off」セクションで日付を選択
  3. 理由（任意）を入力して登録
自動反映: 休みに設定した日は、LINE予約時の空き枠からそのスタッフの枠が自動的に非表示になります。

6. 予約管理

予約の確認

Reservations画面で今週の予約を一覧で確認できます。前週・次週のナビゲーションで週を切り替えられます。

予約ステータスの変更

ステータス意味操作
confirmed予約確定LINE予約時に自動設定
completed来店完了体験後にスタッフが変更 → 自動フォローアップ発動
cancelledキャンセルお客様からのキャンセル時
no_show無断キャンセル来店がなかった場合
重要: 体験レッスン後は必ずステータスを「completed」に変更してください。これが入会案内の自動送信トリガーになります。

手動予約の追加

Reservations画面の「手動予約追加」フォームから、電話予約などLINE以外の予約も登録できます。

7. 顧客管理・会員登録

顧客一覧

LINEから予約したお客様は自動的に顧客一覧に登録されます。名前・メール・電話番号で検索できます。

会員登録（Stripe決済）

  1. 顧客管理画面で、入会させたいお客様のカードアイコン（💳）をクリック
  2. プランを選択
    ・セミパーソナル（通い放題）: ¥19,800/月
    ・マンツーマン: ¥34,800/月
  3. 入会金を選択

    ・通常入会: ¥30,000
  4. 初月合計金額を確認
  5. 「決済リンクを作成」をクリック
  6. Stripe決済ページが新しいタブで開くので、お客様にURLを共有 or その場で入力

10. LINE公式アカウント連携

初期設定（TomorrowProofが実施）

以下はシステム導入時にTomorrowProofが設定します。

  • LINE Developersでチャネル作成
  • Webhook URL設定
  • 管理画面のSettings画面でChannel ID / Secret / Access Token入力

ナレッジの登録

AIが正確に回答するために、ジムの情報をナレッジとして登録してください。

  1. Knowledge画面を開く
  2. 「追加」をクリック
  3. タイトルと内容を入力（例: 「営業時間」「料金プラン」「アクセス」「よくある質問」）
  4. 保存
登録推奨情報:
  • 営業時間・定休日
  • 料金プラン一覧
  • アクセス（住所・最寄駅）
  • 体験レッスンの流れ
  • 持ち物・注意事項
  • よくある質問

11. よくある質問

Q. お客様がLINEで予約したのに通知が来ません

スタッフのLINE User IDが登録されているか確認してください。Schedule → Staff → 該当スタッフの設定で確認できます。管理者への通知は Settings → Escalation User ID に管理者のLINE User IDが設定されている必要があります。

Q. 体験後に自動メッセージが送られません

予約のステータスが「completed」になっているか確認してください。ステータスが「confirmed」のままだと、フォローアップは発動しません。Reservations画面でステータスを変更してください。

Q. 決済リンクが作成できません

Stripe Secret Keyが正しく設定されているか確認してください。テスト環境では「sk_test_」で始まるキーが必要です。Settings画面またはTomorrowProofにお問い合わせください。

Q. AIが正しく回答しません

Knowledge画面に正確な情報が登録されているか確認してください。ナレッジにない質問は「担当者に確認いたします」と回答し、管理者にエスカレーション通知が送られます。

Q. 予約枠が表示されません

Schedule画面で該当曜日に予約枠が登録されているか、またその日が定休日になっていないか確認してください。スタッフの休み設定も影響します。

Q. 複数店舗で使えますか？

はい。店舗ごとにアカウントを作成し、それぞれのLINE公式アカウントと紐づけます。詳細はTomorrowProofにお問い合わせください。
