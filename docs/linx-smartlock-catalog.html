LINXの入退館管理とは

会員がLINEで「チェックイン」するだけでドアが自動解錠。専用端末不要、工事不要、LINEで完結する入退館管理システムです。

会員がLINEで「チェックイン」
→
LINX AIが会員ステータス確認
→
SwitchBotスマートロック解錠
→
10秒後に自動施錠
→
入館記録をダッシュボードに保存
LINE以外でも入退室可能: SwitchBot本体の機能として、暗証番号・指紋認証・NFCカード・顔認証・Apple Watch・Suicaタッチにも対応。LINE + 物理認証の二重対応で、スマホを忘れた会員もアクセス可能。

入退館管理の機能

機能内容
LINEチェックイン会員がLINEで「チェックイン」→ 自動解錠
会員ステータス連動アクティブ会員のみ入館許可。休会・退会は自動ブロック
自動施錠解錠10秒後に自動施錠。閉め忘れ防止
入館ログ日時・会員名・プランをダッシュボードに記録
来館率アクティブ会員に対する来館率をリアルタイム表示
日別来館ログ過去の日付の来館記録も確認可能
物理認証バックアップ暗証番号・指紋・カード・顔認証で LINE不要でも入館可能

導入の流れ

  1. 製品選定 — 施設の要件に合わせてロック+認証パッドを選択
  2. 設置 — ドアに取り付け（工事不要、約15分）
  3. SwitchBotアカウント設定 — APIトークン・シークレットを取得
  4. LINX設定 — ダッシュボードの設定画面でSwitchBot情報を入力
  5. テスト — LINEから「チェックイン」で解錠テスト
  6. 運用開始 — 会員にLINEチェックイン方法を案内
